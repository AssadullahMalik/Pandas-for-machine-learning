# 📊 Pandas Complete Guide 🚀

This repository documents my complete learning journey of **Pandas**, a powerful Python library used for data analysis and manipulation.

---

## 📌 About Pandas

Pandas is widely used in Data Science, Machine Learning, and AI for handling structured data efficiently. It provides fast, flexible, and expressive data structures like Series and DataFrames.

---

## 🧠 Topics Covered

### 🔰 Basics

* Series and DataFrame creation
* Data from lists, dictionaries, and NumPy arrays

### 📂 Data Input/Output

* Reading CSV files
* Writing data to CSV

### 🔍 Data Exploration

* Understanding dataset structure
* Using head(), tail(), info(), describe()

### 🧩 Indexing & Selection

* Selecting rows and columns
* Filtering data
* Using loc[] and iloc[]

### 🧹 Data Cleaning

* Handling missing values
* Removing duplicates
* Filling null values

### 🔄 Data Manipulation

* Adding and modifying columns
* Sorting data
* Applying functions

### 🔗 Grouping & Aggregation

* Using groupby()
* Calculating mean, sum, count

### 🔀 Merging & Joining

* Combining multiple datasets
* Using merge() and concat()

### 📊 Data Visualization

* Basic plotting using Matplotlib
